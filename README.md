# 🐾 Sistema de Refugio de Animales

Aplicación web desarrollada con **Flask y PostgreSQL** para gestionar un refugio de mascotas.

## Funcionalidades

- Login de usuarios
- Gestión de usuarios
- Registro de animales
- Catálogo de adopciones
- Panel de administración

## Tecnologías utilizadas

- Python
- Flask
- PostgreSQL
- HTML
- CSS

2. Instalar dependencias
